# RAG Document Intelligence System

Production RAG pipeline built with LangChain, FAISS, Hugging Face Sentence Transformers, and Groq LLaMA3.

## API Endpoints
- `POST /upload` — Upload a PDF document
- `POST /query` — Ask a question, get a sourced answer
- `GET /docs` — Interactive Swagger UI

## Tech Stack
LangChain · FAISS · Hugging Face Transformers · Groq · FastAPI · Python
